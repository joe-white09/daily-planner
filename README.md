# daily-planner

## Purpose
This is an app that allows you to schedule out your workday.
Insert your objectives for specific hours, while being able to easily 
track how the day is progressing through color coded blocks.

## Built with
* HTML
* CSS
* Javascript
* JQuery
* Bootstrap
